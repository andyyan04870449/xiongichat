## 圖片規格建議
- 格式：JPG 或 PNG
- 尺寸：寬度 800-1200px
- 檔案大小：每張不超過 500KB
- 解析度：72-150 DPI（網頁顯示用）

## 注意事項
1. 所有圖片需確保有合法使用授權
2. 政府宣導圖片優先使用官方提供版本
3. 圖片內容需適合一般大眾瀏覽
4. 避免過於寫實或可能造成不適的圖像
